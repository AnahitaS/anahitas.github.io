2. Understanding the Console:

The RStudio console is where everything happens. There are four fundamental panes in the console. Please follow the learning activity video to learn more about it. Also feel free to check: this tutorial

2.1 Program/Script Pane

The top left Pane is where your .R code file will be shown:

  • To create a new file you use the File -> New File menu.
  • To open an existing file you use either the File -> Open File menu or the Recent Files menu to select from recently opened files. Feel free to explore this pane while following the learning activity video.

To execute the code in the script editor you have two options:

  1. Place the cursor on the line that you would like to execute and execute Ctrl + Enter. OR you could hit the Run button in the toolbar.
  2. If you want to run multiple/all lines of code in the script then you can highlight the lines you want to run and then execute one of the options in #1.

Now try to run your first program. What do you see?

4. Installing & Loading Packages

4.1 Installing Packages

To install packages you typically should use install.packages(packageNname). Example: if you want to install the ggplot2 package, which is a very popular visualization package then type the following in the console:

# install package from CRAN
install.packages("ggplot2")

Important Note: if using VDI, you MUST install packages every single time you log in to VDI.

4.2 Loading Packages

Once the package is downloaded (e.g., to your computer or VDI) you can access the functions and resources provided by using the command library(packageName):

# load the package to use in the current R session
library(ggplot2)
## Warning: package 'ggplot2' was built under R version 4.0.2

Important Note: Unlike install.packages(packageNname) for library(packageName) we do not put packageName inside ..


5. Assignment & Evaluation

An important operator you use is the assignment operator. The assignment operator is used to assign a value. For instance we can assign the value 2 to the variable x using the <- assignment operator.

# assignment
x <- 2
x
## [1] 2

R actually allows for five assignment operators:

# leftward assignment
value <- 5
x <- value
x = value
x <<- value

# rightward assignment
value -> x
value ->> x

Note: for this class I want you to use <-. This is the most conventional assignment operator.

We can evaluate the variable x by simply typing x at the command line which will return the value of x.

# evaluation
x
## [1] 5


6. Be careful about "Case Sensitivity!

R is a case sensitive programming language. This means you MUST pay attention to the exact spelling.

x <- 1 
y <- 3 
z <- 4

x * y * z

# this will raise an error why??
x * Y * z


7. Use of Comments

Note that you can type any comments in your code by preceding the comment with the hash tag (#) symbol. Any values, symbols, and texts following # will not be evaluated.

# this is a comment!
# Assign value 2 to x
x <- 2

Organization of your code is very important. It is impossible to decipher thousands of lines of code that has no organization. The easiest way to achieve organization is to comment your code. I expect that your code has some comments.
